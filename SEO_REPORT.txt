4. IMPLEMENTACJA JĘZYKOWA

Przekierowanie językowe jest zaimplementowane w rootowym pliku index.html na podstawie navigator.language:

<script>
  (function() {
    var lang = navigator.language || navigator.userLanguage;
    if (lang.startsWith('pl')) {
      window.location.replace("./pl/index.html");
    } else if (lang.startsWith('dk') || lang.startsWith('da')) {
      window.location.replace("./dk/index.html");
    } else {
      window.location.replace("./en/index.html");
    }
  })();
</script>

Wniosek:
- Adres URL dla wersji duńskiej jest ustawiony jako "/dk/"
- Tagi językowe dla robotów są ustawione oficjalnie jako "da-DK"

5. ATRYBUTY HREFLANG

Tagi hreflang są obecne w sekcji <head> każdej wersji:

<link rel="alternate" hreflang="pl" href="https://ha-expert.com/pl/" />
<link rel="alternate" hreflang="da-DK" href="https://ha-expert.com/dk/" />
<link rel="alternate" hreflang="en" href="https://ha-expert.com/en/" />
<link rel="alternate" hreflang="x-default" href="https://ha-expert.com/" />

Wniosek:
- Google otrzymuje poprawne mapowanie wersji PL / DA / EN
- Root działa jako x-default
- W sitemap.xml zachowana jest ta sama logika hreflang
